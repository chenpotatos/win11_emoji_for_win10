# **Windows 11 Emoji For Windows 10**

✨让 win10 和 win11 有更一致的 Emoji 体验

## 🧐 食用方法/How to use?


**Step 1** 运行 install.cmd 且不需要以管理员身份运行，运行完成后请重新启动你的设备。

> [!WARNING]
> 
> 如果出现适配性问题，请在重新启动设备后，运行 uninstall.cmd
>
> 这会撤销**所有**对你设备的更改。
